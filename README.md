# CoinScribe
CoinScribe is an all-in-one crypto tracker that shows you not just what the market is doing, but why. It goes beyond simple price charts by integrating a powerful AI engine that instantly reads, understands, and summarizes the latest news for any coin.

## Getting Started

### Prerequisites
- Node.js 18.0.0 or higher
- npm or yarn package manager

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```
This will start the Vite development server on http://localhost:5173

### Debugging

For detailed debugging instructions, see [DEBUG.md](DEBUG.md).

Quick start:
1. Open the project in VS Code
2. Press F5 to start debugging
3. Select "Debug CoinScribe in Chrome" or "Debug CoinScribe in Edge"

### Building for Production
```bash
npm run build
```

### Other Commands
- `npm run lint` - Check code quality
- `npm run lint:fix` - Fix linting issues automatically
- `npm run type-check` - Run TypeScript type checking
